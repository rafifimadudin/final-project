Tentang Model

Informasi detail tentang model machine learning yang digunakan

Ridge Regression Model

Model ini menggunakan algoritma Ridge Regression yang telah dioptimalkan untuk analisis sentimen pada teks media sosial. Model dilatih menggunakan dataset yang terdiri dari 12,000+ postingan dari berbagai platform media sosial.

Keunggulan Model:

Regularisasi L2: Mencegah overfitting dan meningkatkan generalisasi
Feature Engineering: Menggunakan TF-IDF dan metadata untuk prediksi yang akurat
Cross-Validation: Model divalidasi menggunakan 5-fold cross-validation
Hyperparameter Tuning: Alpha optimal (100.0) dipilih melalui RidgeCV

Model Statistics

Training Data: 12,000+ posts
Features: 5,000+ TF-IDF
Alpha: 100.0
